# meridian-migrate
Meridian — open source zero-downtime cross-cloud data migration engine.

PostgreSQL · MongoDB · S3 / GCS / Blob — free forever.

> Migrating data across clouds is painful, manual, and risky. Meridian automates 
> the hardest parts — schema translation, live replication, and parity validation 
> — so you can cut over with confidence and zero downtime.
